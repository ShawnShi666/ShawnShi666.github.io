1. 登陆 mysql

mysql -h服务器主机地址 -u用户名 -p

Eg: mysql -hlocalhost -uroot -p

2. 基本命令行(1)

mysql> show databases;
(查看当前服务器上的所有数据库)

mysql> create database test;
(创建数据库 test)

mysql> use test;
(使用数据库 test)

mysql> show tables;
(查看数据库 test 中的所有数据表)

mysql> drop database test;
(删除数据库 test)

3. MySQL 结构化查询语句分类

名称 解释 命令
DDL(数据定义语言) 定义和管理数据对象，如数据库数据表等 CREATE,DROP,ALTER
DML(数据操作语言) 用于操作数据库对象中所包含的数据 INSERT、UPDATE、DELETE
DQL(数据查询语言) 用于查询数据库数据 SELECT
DCL(数据控制语言) 用来管理数据库的语言,包括管理权限及数据更改 GRANT、COMMIT、ROLLBACK

4. 基本命令行(2)

mysql> create table user(字段信息);
(创建数据表)

mysql> show tables;

mysql> show create table user;

mysql> desc user;

5. 字段属性总结

列类型一般分为：
数值类型、字符串类型、日期和时间型数值类型、NULL 值（不再一一赘述, 不清楚查一下即可.）

列属性：
（1）UNSIGNED:
① 无符号的
② 声明该数据列不允许负数
（2）ZEROFILL:
① 0 填充
② 不足位数的用 0 来填充，如 int(3)，5 则为 005
（3）AUTO_INCREMENT:
① 自动增长的,每添加一条数据,自动在上一个记录数上 + 1
② 通常用于设置主键，且为整数类型
③ 可定义起始值和步长
（4）NULL 和 NOT NULL
① 默认为 NULL，即没有插入该列的数值
② 如果设置为 NOT NULL，则该列必须有值
（5）DEFAULT
① 默认的
② 用于设置默认值

字段 类型 空 默认 注释
StudentNo int(4) 否 学号
LoginPwd varchar(20) 是 NULL
StudentName varchar(20) 是 NULL 学生姓名
Sex tinyint(1) 是 NULL 性别,取值 0 或 1
GradeId int(11) 是 NULL 年级编号
Phone varchar(50) 否 联系电话，允许为空，即可选输入
Address varchar(255) 否 地址允许为空,即可选输入
BornDate datetime 是 NULL 出生时间
Email varchar(50) 否 邮箱账号,允许为空,即可选输入
IdentityCard varchar(18) 是 NULL 身份账号
